NicheHotel Sync integrates hotel bookings and events with Make.com via webhooks, and includes a modern React admin dashboard.

== Description ==
Lightweight plugin to manage hotel data and trigger automation events via webhooks. Includes a SPA admin built with React and Tailwind.

== Installation ==
1. Upload the `nichehotel-sync-ai` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. In the plugin directory, run `npm install` and `npm run build` to compile the admin assets.
